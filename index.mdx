## Quick Start

Get profClaw running in under 5 minutes.

<Steps>
  <Step title="Install">
    ```bash
    # macOS
    brew tap profclaw/tap && brew install profclaw

    # npm
    npm install -g profclaw

    # Docker
    docker pull profclaw/profclaw:latest
    ```
  </Step>
  <Step title="Initialize">
    ```bash
    profclaw init
    profclaw onboard
    ```
  </Step>
  <Step title="Start">
    ```bash
    profclaw serve
    ```
    Open `http://localhost:3000` to access the web UI.
  </Step>
</Steps>

## Key Features

<AccordionGroup>
  <Accordion title="Agentic Execution Engine">
    Multi-step tool execution with self-correction, smart prompts, and sandboxed environments. Supports parallel tool calls and streaming responses.
  </Accordion>
  <Accordion title="50 Built-in Skills">
    Pre-built capabilities like commit, review-pr, deploy, analyze-code, and more. Create custom skills with simple SKILL.md definitions.
  </Accordion>
  <Accordion title="Plugin System">
    Extend profClaw with custom plugins using the SDK. Publish to ClawHub for community sharing.
  </Accordion>
  <Accordion title="Security First">
    Three security modes (permissive, standard, strict), command guards, audit logging, device pairing with QR codes, and plugin sandboxing.
  </Accordion>
  <Accordion title="130+ Environment Variables">
    Fine-grained control over every aspect of the system. All configurable via env vars or settings.yml.
  </Accordion>
</AccordionGroup>
